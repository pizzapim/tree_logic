# Trees Logic Puzzle solver in IDP
This is a (Double) Tree Logic solver, a type of puzzle found on Sporcle by the user Katie_Wandering on Sporcle.
The model is written in IDP, a knowledge base system developed at DTAI, KU Leuven.
The model is for the Double Tree Logic Puzzle, but can be trivially altered to support any number of trees.
